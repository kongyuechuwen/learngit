## 1. 创建版本库

    win CMD
    $ mkdir learngit
    $ cd learngit
    $ pwd

```pwd```命令用于显示当前目录,我的目录是/e/learngit
## 2.初始化版本库

    $ git init
    Initialized empty Git repository in /Users/michael/learngit/.git/

通过```git init```命令把这个目录变成Git可以管理的仓库
